Operating System

I started off with Ubuntu (used it like, 2 days), then switched to Debian for a few months, then used Void for 3 months or so. I was satisfied, Void Linux is a great distribution (and so is Debian btw).

As you know (or maybe you don't) Void Linux uses the runit init system. But later on I needed systemd back because of compatibility issues with an app. I didn't go back to Debian, I installed Arch. Arch is just amazing, AUR, documentation, rolling release (just like Void), and a blazing fast package manager (just like Void). I can, but I didn't went back to Void as I didn't feel the need to.

I thought about switching to Artix but systemd just work fine so...
